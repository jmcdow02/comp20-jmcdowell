Security Assessment of Dan Callahan’s Web Application

by James McDowell, 4/24/15

Introduction

This document assesses the security of Dan Callahan’s Maurader’s Map heroku web application at http://secret-badlands-5595.herokuapp.com/. The app stores records of each user’s login name, location, and check-in time in a MongoDB database. In this report, 3 security vulnerabilities of his application are listed.

Methodology

At first, “black-box testing” was used to primarily assess a few issues, such as cross-site scripting, with cURL.

Next, Dan’s source code was viewed for “white-box testing”, to confirm and identify other suspected security flaws.
